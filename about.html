Voxmo es un projecto para probar la funcionalidad del webspeech API de google.
Al presionar el boton de microfono 🎙️, el navegador recibe audio y lo interpreta, luego utilizando un script de javascript, el usuario es redirigido hacia la pagina indicada por el mismo.
La pagina tambien incorpora diversos efectos de javascript para darle un toque más estetico y pulido.
